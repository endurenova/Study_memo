### 수학 계산을 위한 코드를 제공하는 프로젝트

리모트 레포지토리에서 edit file시 Commit changes에 내용 작성

git pull은 이미 init되어 있는 상태여야만 가능하고 연동 안된 컴퓨터는 clone

git clone https://github.com/numpy/numpy.git: 리모트 레포지토리를 로컬 레포지토리로 복제

텍스트 표시만으로 약속된 시각적 효과를 줄 수 있게 해주는 문법을 마크다운(markdown) 문법 - README.md

git pull: 리모트 레포지토리의 브랜치를 검토할 필요없이 바로 합치고 싶을 때
git fetch: 리모트 레포지토리의 브랜치를 검토해야할 때

로컬 레포지토리를 수정하는 동안 리모트 레포지토리가 수정되서 push가 안된다면 우선 pull
git fetch: 리모트와 로컬 레포지토리가 달라 conflict 발생 상황시 git pull을 사용시 가져옴과 동시에 merge가 되는데 fetch는 merge하지 않고 가져온다 (머지하기 전에 점검해야할 필요가 있을 때 사용)

git merge origin/premium
